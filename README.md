# Project Snapshot Generator

Консольная утилита для создания текстового или Markdown-описания структуры проекта.

## Возможности

* Вывод дерева каталогов и файлов.
* Исключение ненужных директорий (`.git`, `node_modules`, `bin`, `obj` и др.).
* Выгрузка содержимого файлов по выбранным расширениям.
* Автоматическое включение важных файлов проекта:

  * README.md
  * LICENSE
  * .gitignore
  * .editorconfig
* Ограничение максимального размера файлов.
* Экспорт в TXT или Markdown.

## Использование

```bash
python main.py
```

Пример:

```text
Путь к проекту:
D:\Projects\MyApp

Расширения:
.py,.json

Исключить каталоги:
.git,node_modules,venv
```

После выполнения будет создан файл с описанием структуры проекта и содержимым выбранных файлов.

## Для чего это нужно

Инструмент помогает быстро передавать структуру проекта и исходный код в ИИ-ассистенты (ChatGPT, Claude, Gemini и другие) для анализа, ревью или доработки.
